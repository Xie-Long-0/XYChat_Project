# XYChat 安全文档

## M0 当前安全状态

- 客户端对密码执行 SHA-256 摘要后发送给服务端。
- 服务端 SQLite 用户表保存该摘要值，并进行字符串比较。
- 网络通道当前为普通 TCP，尚未启用 TLS。

## 风险

- SHA-256 速度过快，不适合作为生产密码存储方案。
- 摘要在当前协议中等价于登录凭据，被截获后存在重放风险。
- 普通 TCP 无法防止链路监听或中间人攻击。

## 后续要求

- M2 将密码存储升级为随机盐 + 慢哈希 + 参数版本。
- M4 将引入 TLS、证书校验、nonce 和重放保护。
- 日志不得输出明文密码、token、私钥或完整密钥材料。
